.. _examples:

Examples
========

Below are several examples that showcase the different types of jobs you can
run on the Rice HPC resources. All the example code can be found `here <https://github.com/wtbarnes/rice-hpc-examples>`_.
Additionally, you can "clone" the repository of example code to your workspace
on the cluster by running,

.. code-block:: bash

   git clone https://github.com/wtbarnes/rice-hpc-examples.git

at the command line.
